# Environment

En este bloque se incluyen todas las clases utilizadas en la verificación, cuando estas se adjunten deben estar en el siguiente orden jerárquico:

1. `transaction`

2. `generator`

3. `driver`

4. `monitor`

5. `scoreboard`



También se incluyen los mailbox y la interfaz virtual.
